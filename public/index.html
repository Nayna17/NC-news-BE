NC News API Documentation

Welcome to the API reference for my NC News project.

Available Endpoints:

  • GET /api
  • GET /api/topics
  • GET /api/articles
  • GET /api/articles/:article_id
  • GET /api/articles/:article_id/comments
  • POST /api/articles/:article_id/comments
  • PATCH /api/articles/:article_id
  • DELETE /api/comments/:comment_id

    • GET /api

    Returns a list of all available endpoints for the NC News API, along with brief descriptions of their functionality.

    Example Response:

  • "GET /api/topics": Returns a list of topics,
  • "GET /api/articles": Returns a list of articles,
  • "GET /api/articles/:article_id": Returns a single article by ID
